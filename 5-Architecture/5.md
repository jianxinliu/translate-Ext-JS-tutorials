# 架构
Ext JS 遵从MVC/MVVM架构：       
* **MVC——**模型（model） 视图（view） 控制器（controller）  （Ext JS 4.x 版本）   
* **MVVM——**模型（model） 视图（view）视图模型（viewmodel）   （Ext JS 5.x 版本）         
你的程序并不强制要求按照此架构，但按照此架构实现的程序更具有可维护性和易组织性。
